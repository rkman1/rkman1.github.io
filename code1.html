Corn Futures Trade Viability Program

The project involved three components:

  • Reading in a .txt file that contains corn future contracts
  • Distinguish between outright and spread contracts
  • Determine trading relationships between the different markets in sets of three

These were accomplished utilizing:

  • The getLine function placed each corn future into a separate vector
  • If statements were used to classify the futures by type, expiration year and month
    • ZCH0 (ZC = Corn Future, H = March, 0 = 2020 expiration
  • Initializing a different array for each month contract
    • Ex.) ZCH0 = [1,0,0,0,0]
  • Placing each future array into a larger vector
    • Perform linear algebra between each array in the vector and find combinations that sum to zero
