Four Function Clock

Four Functions:

  • Normal clock
  • Alarm clock
  • Light-activated alarm
  • Timer

Clock constructed using the following components:

  • Arduino Uno
  • LCD
    • Displays to the user the time and mode the clock is in
  • Pushbuttons
    • Adjusts light sensitivity for light alarm, increment/decrement timer
  • Potentiometer
    • Cycles between the modes
  • Phototransistor
    • Sensor to detect light levels

The C code consists of four functions: Setup, Loop, and two Interrupts. The code is available to be referenced on my GitHub page.
