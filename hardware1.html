Temperature Sensing Microcontroller

A complete embedded system was constructed that functioned as a thermostat controller with an output display.

Components

  • PIC24 microcontroller
  • DS1631 temperature sensor
  • Two pushbutton switches
  • RGB LED
  • Serial Interface

Functionality

  • The DS1631 sensed air temperate and the PIC24 determined if heating or cooling was needed to keep the air temperature within a user settable range. If cooling was needed (i.e. temperature above the high set point) the LED turned blue, if heating was needed (i.e. temperature below the low set point) the LED turned red, and if the temperature was within the set range the LED turned green.
  • When not being used to set the range, the LCD provided the system state information.
  • The pushbuttons and LCD panel were used to allow a user to input high and low temperature set points. For example, pressing both buttons entered programing mode, first set low set point using one button as up and one button as down. Pressing both buttons again moved to setting the high set point. A third double press exited programming mode and returned the LCD to displaying the state information.
  • The serial port logged all system events.
